[TMHub docs site](http://tmhub.github.io/)

### Installation

```bash
git clone git@github.com:tmhub/tmhub.github.io.git && cd tmhub.github.io

# Install bundler is you don't have it
gem install bundler

# Install dependencies
bundle install
```

### Updating

```bash
bundle update
```

### Runing Locally

```bash
bundle exec jekyll serve
```
